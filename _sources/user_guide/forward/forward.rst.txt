
Generating an AIF
^^^^^^^^^^^^^^^^^

Generate an AIF and plot it:

.. code-block:: python

    import numpy as np
    import matplotlib.pyplot as plt
    import dcmri

    t = np.arange(0, 6*60, 1)
    ca = dcmri.aif_parker(t)
    plt.plot(t, ca)
    plt.show()
